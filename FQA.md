# 📚 RayBot 常見問題與解答 (FAQ)

## 🟢 壹、一般概念問題

### Q1：為什麼我要自己架設 (Self-Hosted)？
**A：**
1.  **絕對隱私**：資料只存在您的 Cloudflare 帳號，開發者無法查看。
2.  **資料自主**：不用擔心服務倒閉資料消失。
3.  **成本控制**：利用 Cloudflare 免費額度，個人使用幾乎零成本。

### Q2：我可以一個機器人給多個群組使用嗎？
**A：**
可以。系統支援「多租戶」，不同群組的資料與設定是完全獨立的。

## 🟡 貳、架設與故障排除

### Q1：LINE Webhook Verify 失敗？
**A：**
1.  確認 Worker 已 Deploy 且狀態為 Active。
2.  確認網址以 `https://` 開頭。
3.  確認 Super Admin 未開啟「後端維護模式」。

### Q2：資料庫出現 `no column named...` 錯誤？
**A：**
這是資料庫結構未同步。本系統內建自動修復，請在 LINE 群組發送任意訊息，或重新整理網頁即可自動補全欄位。

### Q3：後台顯示 "Unknown Action"？
**A：**
通常是前後端版本不一致或程式碼複製不完整。請重新部署完整的 `worker.js` 並清除瀏覽器快取 (`Ctrl + F5`)。

## 🔵 參、日常操作問題

### Q1：輸入作業（如：`明天考英文`）沒抓到？
**A：**
1.  **日期**：必須包含明確時間詞 (明天、下週一)。
2.  **科目**：檢查後台「科目設定」是否有該科目的關鍵字。
3.  **AI**：若依賴 AI，請確認該群組 AI 權限未被鎖定，且後台已開啟 AI。

### Q2：`/bot agree` 沒反應？
**A：**
若已同意過，系統不會重複提示。若需測試，可輸入 `/bot reboot` 重置同意狀態。

## 🔴 肆、進階與安全

### Q1：密碼安全嗎？
**A：**
系統採用 **PBKDF2 (600,000 次迭代)** 加密，並為每個密碼加鹽 (Salt)，符合 NIST 安全標準。

### Q2：什麼是「操作密碼」？
**A：**
在 Super Admin -> 設定中可開啟。開啟後，執行「刪除群組」或「重置狀態」等危險動作時，需再次輸入此密碼驗證。

### Q3：如何防止別人按 F12？
**A：**
在 Super Admin 設定中勾選 **「🚫 禁用開發者工具」**。這會在前端網頁注入腳本，阻擋右鍵與 F12 快捷鍵。

### Q4：救援碼 (6碼) 與 復原碼 (10碼) 的差別？
*   **救援碼 (Rescue Code, 6碼數字)**：用於一般後台「忘記密碼」時重置密碼，通常由總管理員持有。
*   **復原碼 (Restore Code, 10碼亂數)**：系統生成的唯一識別碼，通常用於 Super Admin 搜尋或未來的進階救援，使用者平常不會用到。
